# testing444

A simple React dashboard built with [Recharts](https://recharts.org/).

## Files

| File | Description |
|------|-------------|
| `dashboard.jsx` | Reusable `<Dashboard>` line-chart component |
| `sampleData.js` | Example monthly data consumed by the dashboard |
| `App.jsx` | Root component that wires `Dashboard` + data together |
| `index.js` | Entry point – mounts the React app onto `#root` |

## Getting started

```bash
npm install react react-dom recharts
# then bundle / serve with your preferred tool (Vite, CRA, etc.)
```
